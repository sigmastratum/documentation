# Sigma IP, Licensing, and Certification Policy

| Field | Value |
|---|---|
| Version | 3 / 2026-05-23 |
| Replaces | Sigma Stratum Canon IP Framework |
| Maintainer | Sigma Stratum Research Group |

## Core Position

The Sigma Runtime Standard (`SRS`) and accepted Sigma Runtime Improvement Proposals (`SRIPs`) are open for independent implementation, including commercial implementation.

No Sigma commercial runtime license is required solely because an independent implementation follows the public SRS/SRIP normative requirements.

SRS/SRIP is the public specification layer. Sigma Runtime as an operated product remains proprietary.

Product assets, marks, certification, deployment rights, and restricted research materials are governed separately.

Independent implementation is permitted. Official certification, endorsement, and Sigma mark use require separate approval.

## What Is Open

| Layer | Public posture | License / policy |
|---|---|---|
| SRS/SRIP public specification | Open for independent implementation | CC BY 4.0 + Independent Implementation Safe Harbor |
| Machine-readable public artifacts | Open where explicitly marked | Apache 2.0 where marked |
| Conformance fixtures / validators | Open where explicitly marked | Apache 2.0 where marked |
| Research corpus | Per-document license | CC BY 4.0 or CC BY-NC 4.0 depending on document |

## What Uses Its Own Terms

| Layer | Policy boundary |
|---|---|
| Sigma Runtime product assets | Product terms / commercial license |
| ALICE internals, telemetry, private modules, runtime implementation, product UX, deployment topology | Proprietary product boundary |
| Sigma marks, `∿`, official badges, official compatibility language | Marks and Certification Policy |
| Official certification and registry status | Certification Policy and registry listing |
| Managed deployment, white-label deployment, resale | Commercial runtime terms |
| Commercial use of CC BY-NC materials | Separate authorization |

The protected Sigma symbol is `∿` (Unicode `U+223F`, Sine Wave). ASCII tilde (`U+007E`) is not the Sigma mark.

## Implementation Safe Harbor

Independent implementation of public SRS/SRIP normative requirements is permitted under the applicable public specification terms.

Builders may study, implement, publish, and sell independent implementations that follow the public normative requirements.

No Sigma commercial runtime license is required solely because an implementation follows those public normative requirements.

## Patent Notice

No patent rights are granted unless explicitly stated in a separate public specification covenant, patent non-assertion commitment, or written agreement.

## Bounded Multiplicity DOI Distinction

Protected CC BY-NC research-origin materials include Bounded Multiplicity:

| Citation role | DOI | Meaning |
|---|---|---|
| Concept DOI | [10.5281/zenodo.20325364](https://doi.org/10.5281/zenodo.20325364) | General citation for the work family |
| Version DOI | [10.5281/zenodo.20325365](https://doi.org/10.5281/zenodo.20325365) | Specific archived version reviewed for licensing or compliance |

Use the concept DOI for general citation. Use the version DOI when citing the exact archived version reviewed for licensing or compliance.

## Canon Terminology

Earlier Sigma Stratum documents used "Canon" for the IP and licensing framework. The active public-facing policy is this document. "Canon" is retained only as historical / retired terminology.

For active public-facing licensing interpretation, this policy supersedes prior Canon/CIL terminology unless a historical document is being cited as archival material.
