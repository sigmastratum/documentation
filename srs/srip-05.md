# SRIP-05 - Interoperability Interface v1.0

**Status:** Draft  
**Category:** Interoperability  

This SRIP specifies the interoperability layer of Sigma Runtime, including:

- state exchange schemas  
- attractor metadata interoperability  
- drift signal interfaces  
- external memory integration  
- versioning rules  

This interface ensures that independent systems adhering to the standard remain
interoperable.
